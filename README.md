# Execution Instructions

## Deploy
`./deploy.sh <deploy-directory>`
## Run server
`<deploy-directory>/server/run-server.sh`
## Run clients

`<deploy-directory>/client/query1 -Dcity=C -Daddresses='xx.xx.xx.xx:XXXX;yy.yy.yy.yy:YYYY' -DinPath=XX -DoutPath=YY`
